## Getting Started

These instructions will get you a copy of the project up and running on your local machine.

### Prerequisites

### Installation

    ```bash
    git clone [invalid URL removed]
    ```

2.  Navigate to the project directory:

    ```bash
    cd your-todo-list
    ```

3.  Open the `index.html` file in your web browser.

## Usage

1.  **Adding a task:** Type your task in the input field and press Enter or click the "Add" button.
2.  **Marking a task as complete:** Click the checkbox next to the task.
3.  **Deleting a task:** Click the "Delete" button next to the task.
4.  [Optional: Editing a Task: Double click on the task to edit it.]
5.  [Optional: Using Local Storage: The tasks will be saved in your browser's local storage and will be available even after you close and reopen the browser.]
